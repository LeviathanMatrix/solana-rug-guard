 Security Policy

 Reporting a Vulnerability

If you believe you have found a security vulnerability, please do **not** open a public GitHub issue.

Instead, report it privately:

 Email: `security@yourdomain.com`
 Subject: `Security report: Solana Rug Guard`

Please include:
 a clear description of the issue and impact
 steps to reproduce (or a proof of concept)
 affected components and versions (if known)

We will acknowledge receipt and work with you on a responsible disclosure timeline.
